## 为什么自建博客？

之前在 CSDN 上写了不少技术文章，但总感觉不太自在：

- **平台绑定** — 内容在别人手上，哪天改规则了只能认
- **广告泛滥** — 打开自己写的文章，满屏广告
- **编辑体验差** — Markdown 支持不完善，排版费劲

所以决定迁移到 **Hugo + GitHub Pages**：

| 对比项 | CSDN | Hugo + GitHub Pages |
|--------|------|---------------------|
| 数据归属 | 平台 | 自己（本地 + Git） |
| 广告 | 满屏 | 零 |
| 编辑 | 在线编辑器 | 本地 Markdown + Git |
| 定制 | 有限 | 完全自由 |
| 费用 | 免费（用流量换） | 免费 |

## 关于"温陵布衣"

**温陵**是泉州的古称，离我所在的厦门不过百里；**布衣**取"布衣之士"，意指一介书生。

名字的寓意很简单：**身处闽南，以技术为业的一介普通人**。不装大牛，不追热点，踏踏实实写点东西。

## 技术栈

- **框架**：[Hugo](https://gohugo.io/) — 极速静态站生成器
- **主题**：[PaperMod](https://github.com/adityatelange/hugo-PaperMod) — 简洁优雅
- **部署**：GitHub Pages + GitHub Actions 自动构建
- **写作**：Markdown + Git 版本管理

- 嵌入式 Linux / Android 开发实践

不追求更新频率，有干货才写。
